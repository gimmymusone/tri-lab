# TRI LAB — Innovation Hub

Sito one-page dell'agenzia **TRI LAB**: ingegneria gestionale, economia e AI generativa per le PMI. Creative, Business e Operation Hub.

Sito statico self-contained (HTML/CSS/JS inline, GSAP + Three.js da CDN). Pagine di approfondimento per i tre hub in `tre-hub/`.

## Struttura
- `index.html` — home
- `logo.svg` — logo (3 triangoli intrecciati)
- `tre-hub/` — pagine hub (`*-hub.html`), pagina servizio (`servizio.html`), dati (`hub-data.js`), stile/JS condivisi (`hub.css`, `hub.js`)
- `Anteprime/web/` — video anteprima dei siti realizzati (mp4 web-ottimizzati)
- `ChiSiamo/` — foto del team
